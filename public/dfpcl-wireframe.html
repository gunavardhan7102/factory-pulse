System Notes & Architecture
Multi-Plant Hierarchy
Plants: • Taloja (Primary): Ammonia, Nitric Acid, Utility • Ratnagiri: Ammonia, Utility • Goa: Utility-focused Equipment per Plant: Taloja: 40 assets Ratnagiri: 20 assets Goa: 10 assets
Equipment Types
• Compressors • Reactors • Heat Exchangers • Pumps & Towers • Tanks & Boilers • Chillers & Coolers • Control Valves • Agitators
Sensor Metrics
• Temperature (°C) • Vibration (mm/s) • Pressure (kPa) • Flow Rate (m³/h) • Tank Level (%) • pH Value • Bearing Temp (°C) • Differential Pressure
Equipment Status
Operating Status: • Running • Standby • Maintenance • Shutdown Health Status: • Healthy (Green) • Warning (Yellow) • Critical (Red)
Key Features Across All Roles
Manager Access: Dashboard, Fleet View, Asset Details, Digital Twin, Maintenance Tracking, Analytics, Connectivity, Alerts, AI Copilot
Technician Access: Mobile Workspace, Work Orders, Inspections, Real-time Sensors, Photo Documentation, Offline Support, Task Tracking
Admin Access: User Management, System Settings, Plant Configuration, Alert Rules, Audit Logs, Data Management, Integration Settings
